##  Projects

### 1.  Student Sleep Hours Prediction
- **Dataset:** Student lifestyle data
- **Goal:** Predict how many hours a student sleeps
- **Steps:** Data cleaning, EDA, Model Building
- **Models:** Linear Regression, KNN, Random Forest
- **Best Model:** Random Forest
- **Key Findings:** Stress level, study hours and screen time are the biggest factors

### 2.  Diabetes Risk Prediction
- **Dataset:** 6,000 patient records, 19 features
- **Goal:** Predict diabetes risk in patients
- **Steps:** Data cleaning, EDA, Outlier handling, Encoding, Model Building
- **Models:** SVM, KNN, Decision Tree, Random Forest
- **Best Model:** Random Forest
- **Key Findings:** HbA1c level is the strongest predictor of diabetes risk
