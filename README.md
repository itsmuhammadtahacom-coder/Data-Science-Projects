## Projects

### 1. Student Sleep Hours Prediction
- **Dataset:** 4,998 student records, 19 features (lifestyle & academic data)
- **Goal:** Predict how many hours a student sleeps per night
- **Steps:** Data cleaning, missing value imputation, EDA, feature scaling, model building
- **Models:** Linear Regression, KNN, Random Forest
- **Best Model:** Linear Regression (RMSE: 1.447, R²: -0.009)
- **Key Findings:** Sleep hours showed low linear predictability from lifestyle features alone

### 2. Diabetes Risk Prediction
- **Dataset:** 6,000 patient records, 19 features
- **Goal:** Classify patients into Low Risk, Prediabetes, or High Risk categories
- **Steps:** Data cleaning, EDA, outlier handling, encoding, feature scaling, model building
- **Models:** SVM (88.4%), KNN (86.8%), Decision Tree (100%), Random Forest (99.9%)
- **Best Model:** Random Forest (Accuracy: 99.9%)
- **Key Findings:** HbA1c level is the strongest predictor of diabetes risk

### 3. E-Commerce Purchase Prediction
- **Dataset:** 1,500 customer records with age, income, spending score, device, city tier
- **Goal:** Predict whether a customer will make a purchase (binary classification)
- **Steps:** Data cleaning, missing value imputation, EDA, encoding, feature scaling, model building
- **Models:** SVM, Random Forest, KNN
- **Best Model:** Random Forest (Accuracy: 70.2%)
- **Key Findings:** Spending score and annual income are the strongest purchase predictors
